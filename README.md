# sales-profitability-Optimization- analysis
Sales Profitability Optimization

Problem

Despite high sales, the company's profit was decreasing. The goal of this project was to identify the key factors affecting profitability.

Objective

Analyze retail sales data to understand product performance, regional profitability, and the impact of discounts.

Tools Used

- Excel
- SQL
- Python (Pandas, Matplotlib)
- Power BI

Analysis Steps

1. Data cleaning and preprocessing using Python
2. SQL queries to analyze product performance and regional sales
3. Data visualization and dashboard creation using Power BI

Key Insights

- High discounts significantly reduced profit margins
- Some regions generated consistent losses
- Certain products had low profitability despite high sales
